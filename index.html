HTML

시멘틱 태그에 대한 깊은 이해를 바탕으로 직관적이고 접근성 높은 구조를 작성할 수 있으며, 유지보수가 쉽고 협업에 최적화된 코드를 구현할 수 있습니다

CSS

속성과 값에 대한 이해를 바탕으로 기획한 디자인을 구현하며, 다양한 해상도에 대응하는 반응형 웹과 애니메이션을 활용한 동적인 사용자 경험을 제공합니다

SCSS

중첩 구조로 정확한 태그 경로를 신속하게 지정할 수 있으며, 변수와 믹스인을 통해 코드 재사용성과 유지보수 효율성을 높입니다.

JavaScript

표준 문법을 준수하며, ES6+을 활용해 가독성과 유지보수성이 높은 코드를 작성할 수 있습니다. 이벤트 처리를 통해 동적인 웹 인터페이스를 구현합니다

jQuery

애니메이션, DOM 조작, 이벤트 관리를 효율적으로 구현할 수 있으며, 간결하고 최적화된 코드를 작성하여 작업의 완성도를 높일 수 있습니다.

React

컴포넌트 기반의 재사용 가능한 UI를 효율적으로 작성하며, 데이터 변화에 따라 동적으로 렌더링을 처리해 유지보수성이 높은 웹 애플리케이션을 구현합니다

Figma

스타일 가이드를 통해 일관성 있는 UI 디자인을 구현할 수 있으며, 오토 레이아웃을 활용해 디바이스 크기에 유연하게 대응합니다

Photoshop

이미지 편집, 보정, 레이아웃 디자인등 다양한 작업을 수행하며, 구상한 아이디어를 능숙하게 구현해 완성도 높은 결과물을 제작할 수 있습니다

Illustration

벡터 기반의 작업을 통해 로고 디자인, 아이콘 제작, 인포그래픽 등 다양한 시각 자료를 제작할 수 있는 역량을 보유하고 있습니다
